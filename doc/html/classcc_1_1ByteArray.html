bool cc::ByteArray::toUtf16 ( void *  buf,
int *  size 
)

Returns true if the given buffer was suitable to hold the encoded string. The number of bytes required to fully represent the string in UTF-16 is returned with the 'size' argument. Passing a zero for 'size' allows to determine the required buffer size. No zero termination is written or or accounted for.
