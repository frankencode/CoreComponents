Detailed Description

template<class T>
class cc::Timer< T >

A Timer is a thread that writes tick objects into a channel in a fixed time interval. The time interval may not be hit exactly for each tick, but the overall time scale is maintained. In other words an individual tick may be delivered unprecisely due to the uncertainty of thread scheduling, but there is no error accumulated in the process.
