24 /** \brief Duck-typed value
25  *
26  * A Variant can represent different types.
27  * The type of a Variant is defined implicitly at construction or assignment.
28  * Variants automatically cast to bool, int, float if the Variant type is compatible
29  * with the target type requested by an expression.
30  * In debug mode a DebugException will be thrown on illegal type casts.
31  * Variants can be savely passed by value.
32  */
