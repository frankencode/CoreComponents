Detailed Description

Base class for all classes T, whose instances can be referred to by Ref<T>. Enforces a consistent allocation schema by surpressing two things:

  • combination of static allocation and dynamic destruction
  • manual detruction by delete operator In debug mode in both cases a DebugException is thrown.
