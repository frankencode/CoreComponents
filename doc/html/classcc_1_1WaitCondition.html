void cc::WaitCondition::wait ( Mutex *  mutex)

The thread will be woken up again and reaquire the mutex atomically if the availability of the condition is signalled (to an arbritary number of threads by signal() or all threads by broadcast()). Note that the first thread scheduled by the OS may invalidate the condition again.
